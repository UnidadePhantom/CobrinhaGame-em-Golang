# Jogo da Cobrinha em Go (Ebiten)
![Go](https://img.shields.io/badge/GameSnake-Go-lightblue?logo=go)
## 📖 Descrição
Projeto desenvolvido em **Go (Golang)** utilizando a biblioteca **Ebiten** para renderização gráfica.  
Este jogo da cobrinha foi criado como exercício prático de lógica de programação, controle de fluxo e manipulação gráfica em tempo real.

---

## 🛠️ Tecnologias
- Linguagem: Go (Golang)
- Biblioteca gráfica: Ebiten
- Paradigma: Programação estruturada e lógica de jogos
- Conceitos aplicados: game loop, leitura de teclado, controle de estado, renderização 2D

---

## 🎯 Objetivo
- Demonstrar capacidade de implementar jogos clássicos em Go.  
- Explorar conceitos de **game loop** e **entrada de usuário**.  
- Mostrar aplicação prática de Go em desenvolvimento de jogos 2D.
